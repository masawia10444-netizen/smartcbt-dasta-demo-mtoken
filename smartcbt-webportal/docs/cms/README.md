# CMS Integration

## Overview

The CMS integration allows you to collect and enrich your CMS logs.

## Setup

### Installation

The CMS check is included in the [Datadog Agent][1] package, so you don't need to install anything else on your CMS servers.

### Configuration

1. Edit the `cms.d/conf.yaml` file, in the `conf.d/` folder at the root of your [Agent's configuration directory][2] to start collecting your CMS performance data. See the [sample cms.d/conf.yaml][3] for all available configuration options.
